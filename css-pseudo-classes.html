CSS Pseudo-classes

A pseudo-class is used to define a special state of an element.For example, it can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

Link Link Link Link Custom Link Custom Link 

/* basic css link styling */ 

	a:link{color:#222121;text-decoration:none} /*styles the link text before it has been clicked */

	a:visited{color:#222121;text-decoration:none} /*styles the link text after it has been clicked */

	a:hover{color:#0023F1;text-decoration:underline;} /*styles the link text when cursor is over link text */

	a:link.customlink{color:#ff0000;text-decoration: none;} /*styles the link text with the class=customlink before it has been clicked */

	a:visited.customlink{color:#ff0000;text-decoration: none;} /*styles the visited link text with the class=customlink after it has been clicked */

	a:hover.customlink{color:#00ff00;text-decoration:underline;}/*styles the link text with the class=customlink when cursor is over link text */
